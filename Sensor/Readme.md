# Датчик влажности почвы - Soil moisture sensor

Датчик проверялся в воздухе и в слегка подсоленой воде. Датчик имеет следующие характеристики:
  * Емкость между электродами
    * в воздухе - 7.7 pF
    * в воде - 130 pF
  * Частота на выходе счетчика, R3 = 1 kOhm 
    * в воздухе - 13.0 kHz
    * в воде - 2.2 kHz
